# 🌍 SimpleQuiz

A fun console quiz in Swift.  
Test your knowledge of world capitals.

## 🎯 Features

- 🧠 Questions about capitals
- ✅ Correct answer counter
- ❌ Shows the right answer if you're wrong
- 🚪 Exit command

## 🗺️ Example
start
Quiz started
Capital of France? Paris
✅ Correct!
Capital of Great Britain? London
✅ Correct!
Capital of China? Peking
❌ Wrong. Correct answer: Beijing
...
🏆 Your score: 7 out of 9

exit
👋 Bye!

text

## 📝 Commands

| Command | Action |
|---------|--------|
| `start` | start the quiz |
| `exit` | exit the program |
